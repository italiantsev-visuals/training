# Программа тренировок

**Обновлено**: 2026-05-27 | **Данные**: Strong (апрель 2025 – май 2026, 104 сессии) + Strava (кардио интегрировано в dashboard)

---

## Профиль

| Параметр | Значение |
|----------|----------|
| Вес | 83 кг |
| BF | ~15–16% |
| Цель | Body recomposition: −2–3 кг жира + грудь/руки/подтягивания |
| Локации | Домашний зал (гантели 50lb, кабель, leg ext/curl) · LA Fitness (нерег.) · Дом (турник) |
| Кардио | Ролики / велосипед / ходьба (бег пока тяжело) |
| Реальная частота | 2–3 тренировки/нед (пик — 3.2, реальная — 2–2.5) |
| Минимальная неделя | 2 × 30–50 мин (аварийный режим) |

---

## Структура: база + ротация

**Каждая сессия = БАЗА + РОТИРУЮЩИЙ БЛОК**

```
БАЗА (обязательно):
  Грудь (2–3 упражнения)
  Спина (2–3 упражнения)
  Lateral Raise 3×15       ← обязательно, не по остаточному принципу
  Face Pull / Reverse Fly  ← обязательно, здоровье плеч

РОТИРУЮЩИЙ БЛОК (чередуется):
  Сессии 1 и 3 → Суперсет: Bicep Curl / Hammer Curl → Triceps Pushdown / Extension
  Сессии 2 и 4 → Предплечья (Reverse Curl + Dead Hang) + Кор (Cable Crunch)

ОТДЕЛЬНЫЕ СЕССИИ:
  Ноги → 1×/нед, 30 мин · Leg Ext + Leg Curl + RDL + Split Squat
  Аксессуары (икры + что добавишь) → можно совместить с ногами
```

**Ноги** — отдельная мини-сессия 1 раз в 2 недели (30–40 мин), не смешивать с базой.

**Вариант Б — Дома** (гантели, кабель, leg ext/curl machine)

| # | Упражнение | Подходы×Повт | Отдых | Рабочий вес → цель |
|---|-----------|-------------|-------|-------------------|

> Ноги **1×/нед** — научный минимум для maintenance при твоей базе. Совместить с кардио: 20 мин велик → 40 мин ноги.

## Правила отдыха между подходами

| Тип упражнения | Отдых | Примеры |
|---------------|-------|---------|
| Тяжёлые компаунды | **2–3 мин** (или пока HR < 100) | Bench, Pull Up, Row, Lat Pulldown |
| Лёгкие компаунды | **90 сек** | Single Arm Row, Cable Fly, RDL |
| Изоляция | **60–75 сек** | Lateral Raise, Curl, Triceps, Face Pull |
| Суперсеты | **60 сек после пары** | Curl → Pushdown |
| Предплечья / Dead Hang | **45 сек** | |

> Ждать пока пульс упадёт до 90–100 перед тяжёлыми сетами — правильная интуиция, следуй ей.

## Аварийный режим (загруженная неделя)

**2 сессии** — этого достаточно чтобы не откатываться. Два варианта:

### Вариант A — минимальный (~30 мин)
Когда совсем мало времени или высокий стресс:
```
Bench Press или Cable Press  3×10
Lat Pulldown                 3×12
Lateral Raise                3×15
Face Pull                    3×20
────────────────────────────────
4 упражнения × 3 подхода = ~30 мин
```

### Вариант B — плотный (~40 мин, рекомендую)
Когда есть 40 мин — чуть больше вариативности, чуть лучший стимул:
```
Incline Bench или Cable Press  2×10
Cable Fly                      2×13    ┐ суперсет с ↓
Reverse Fly                    2×15    ┘
Lat Pulldown                   2×12
Seated Row                     2×12
Lateral Raise                  2×15
────────────────────────────────────────
6 упражнений × 2 подхода = ~35–40 мин
```

> **Наука**: 2 подхода ≈ 70–80% эффективности от полного объёма. Переход с 0 сетов к 2 сетам даёт основной прирост — дополнительные сеты приносят убывающую отдачу. 2 дня × 2 сета каждый = удержание прогресса гарантировано.

> Оба варианта работают — Вариант B чуть лучше за счёт бо́льшего разнообразия движений при том же времени.

## Кардио

| Тип | Частота | Пульс | Длительность |
|-----|---------|-------|-------------|
| Велик / ролики / ходьба | 1–2×/нед | 130–145 | 45–60 мин |
| Лёгкий бег | по желанию | 130–140 | 20–30 мин |

> Не делать тяжёлое кардио (HR > 155) за 24 ч до или после тяжёлой силовой.
> Лёгкое кардио на следующий день после зала — ускоряет восстановление, не мешает.

## Питание (ориентиры)

- Дефицит: **~10–15%** от TDEE (лёгкий — мышцы не теряются)
- Белок: **≥150 г/день** при 83 кг (текущий режим — правильный)
- До видимого нижнего пресса: **−2–3 кг жира** (с ~15–16% до ~12–13% BF)
