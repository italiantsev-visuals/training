# Правила тренировок

**Вес**: 83 кг | **BF**: ~15-16% | **Цель**: body recomposition (−2–3 кг жира + грудь/руки/подтягивания)

---

## Базовые принципы

- **Частота**: реальный уровень ~2 тренировки/нед, пик ~3. Постоянные 2x/нед > нерегулярные 4-5x
- **Прогрессия**: каждую неделю +1 повтор ИЛИ +2.5 lb в одном подходе — этого достаточно
- **RPE / RIR**: рабочие подходы — RIR 2-3, последний подход — до отказа или RIR 1
- **Структура**: каждая сессия = **БАЗА** (грудь + спина + lateral raise + face pull) + **РОТИРУЮЩИЙ БЛОК** (трицепс / бицепс / суперсет)

---

## Структура: база + ротация

**БАЗА — в каждой тренировке:**
1. Грудь — 2-3 упражнения (Incline Bench первым — свежая мышца на верх груди)
2. Спина — 2-3 упражнения (вертикальная + горизонтальная тяга)
3. **Lateral Raise 3×15** — ОБЯЗАТЕЛЬНО, не по остаточному принципу
4. **Face Pull / Reverse Fly 3×20** — ОБЯЗАТЕЛЬНО, здоровье плеч

**РОТИРУЮЩИЙ БЛОК** (чередуется каждую сессию):
- Сессия 1 → Трицепс 3×15
- Сессия 2 → Бицепс 3×12 + Предплечья (Dead Hang + Reverse Curl)
- Сессия 3 → Три+Бис суперсет 3×(12+15)
- Сессия 4 → повтор с сессии 1

**НОГИ — отдельная мини-сессия:**
- 1 раз в 2 недели, 30-40 мин
- **Никогда не смешивать с базовой сессией** — CNS overload от трёх больших групп (грудь + спина + ноги)
- Можно совместить с кардио-днём: 20 мин лёгкий велик → 30 мин ноги

**Состав ноги:**
1. Leg Extension 3×12 (рабочий вес 180-200 lb)
2. Seated Leg Curl 3×12 (рабочий вес 150 lb)
3. RDL с гантелями 3×12 (45 lb — хороший вес)
4. Bulgarian Split Squat 3×10 (если нет поясничного дискомфорта)

---

## Правила отдыха между подходами

| Тип упражнения | Отдых | Примеры |
|---------------|-------|---------|
| Тяжёлые компаунды | **2-3 мин** (или пока HR < 100) | Bench Press, Pull Up, Row, Lat Pulldown |
| Лёгкие компаунды | **90 сек** | Single Arm Row, Cable Fly, RDL |
| Изоляция | **60-75 сек** | Lateral Raise, Curl, Triceps, Face Pull |
| Суперсеты | **60 сек после пары** | Curl → Pushdown, без отдыха между упражнениями |
| Предплечья / Dead Hang | **45 сек** | |

> Ждать пока пульс упадёт до 90-100 перед тяжёлыми сетами — правильная интуиция, следуй ей.

---

## Суперсеты и предплечья

**Суперсет грудь + задняя дельта (финишёр):**
- Формат: Cable Fly → Reverse Fly, 3×(13+15)
- Делается ПОСЛЕ компаундов (жимов и тяг), не до — свежие мышцы нужны для тяжёлых движений
- Fly: пауза 2 сек в сведении; Reverse Fly: лёгкий вес, качество

**Антагонист-суперсет (бицепс + трицепс):**
- Формат: Bicep Curl → Triceps Pushdown, 3×(12+15)
- Переход между упражнениями без отдыха, 60 сек после завершения пары
- Вариант: Hammer Curl → Triceps Extension

**Предплечья (добавлены как слабое место):**
- Dead Hang 3×20-30 сек — предплечья + растяжка широчайших
- Reverse Curl / Wrist Curl 2×15 — прямая нагрузка
- Включаются в сессию с бицепсом (сессия 2 и 4)

## Ключевые технические правила

### Жим груди — почему болят плечи
- Думай **"свожу локти/бицепсы"**, НЕ "толкаю вес вверх"
- Пауза 1 сек в нижней точке жима штанги/гантелей
- Face Pull в конце КАЖДОЙ тренировки — снимает нагрузку с передней дельты
- Если плечи ноют во время жима — снизь вес на 15-20%, добавь паузу

### Подтягивания — как прогрессировать
- Резинка лёгкая: 3×8-10 с RIR 2-3 >> 3×5-6 до отказа без резинки
- НЕ до отказа первые 2 подхода — иначе третий будет 5 вместо 8
- Bicep Curl 3×12 в конце тренировки со спиной — слабый бицепс лимитирует подтягивания

### Прогрессия при ограниченном весе (50 lb гантели дома)
- Вес ограничен → компенсируй повторениями (12-15 вместо 8-10)
- Пауза 2 сек в нижней точке = эффективнее чем просто добавить вес
- Отдых сокращай до 45 сек между изоляционными подходами

## Вкатывание после перерыва

| Фаза | Недели | Веса | Интенсивность |
|------|--------|------|---------------|
| Адаптация | 1-2 | −20% от рабочего | Без отказных. RIR 3-4. |
| Возврат | 3-4 | Рабочие веса | Последний сет RIR 1-2. |
| Прогрессия | 5+ | +1 повт или +2.5-5 lb/нед | Последний сет до отказа или RIR 1. |

## Объёмы — норма и реальность

| Мышца | Сетов/нед | Норма (MEV→MAV) | Статус |
|-------|----------|-----------------|--------|
| Грудь | ~10 | 8→20 | ✅ |
| Спина | ~10 | 8→25 | ✅ |
| Плечи средние (Lateral Raise) | ~6 | 6→16 | ✅ теперь каждая сессия |
| Плечи задние (Face Pull) | ~6 | 4→16 | ✅ теперь каждая сессия |

| Трицепс (прямые + жимы×0.5) | ~7-8 | 4→20 | ✅ |
| Предплечья | ~3-4 прямых | — | ✅ добавлены |
| Ноги | 1×/2 нед + кардио | — | ✅ |
| Кор | нерегулярно | 8→16 | ⚠️ по желанию |

## Локации и стратегия

| Локация | Когда использовать | Лучшие упражнения |
|---------|--------------------|-------------------|
| Домашний зал | основные тренировки (2x/нед) | кабельные жимы/тяги, гантели до 50 lb, изоляция |
| LA Fitness | нерегулярно — тяжёлые базовые | штанга, hack squat, leg press, T-bar row |
| Дом | поддержание / экстренный режим | подтягивания на турнике |

**Стратегия LA Fitness**: используй для штанги (Bench 155-165 lb, Incline 115 lb, T-bar row) — дома не повторишь.

## Аварийный режим (загруженная неделя)

**2 сессии × 30-40 мин** — этого достаточно чтобы не откатываться:

```
Bench Press или Cable Press  3×10
Lat Pulldown                 3×12
Lateral Raise                3×15
Face Pull                    3×20
────────────────────────────────
~30-35 мин, всё главное закрыто
```

---

## Кардио

- Ролики / велик: пульс **130-145** — жиросжигание без потери мышц
- Выше 155 пульс — стресс для восстановления, избегать за 24ч до/после тяжёлой силовой
- Лёгкое кардио на следующий день после зала — ускоряет восстановление, не мешает
- 1 час роликов при 150 пульс = отдельный тренировочный день, учитывай при планировании

## Питание

| Показатель | Значение |
|------------|----------|
| TDEE | ~2823 ккал/день |
| Цель (дефицит 12%) | **2470 ккал/день** |
| Белок (Б) | **166 г** (2 г/кг) |
| Жиры (Ж) | **70 г** (25% ккал) |
| Углеводы (У) | **295 г** |

- Дефицит лёгкий — мышцы не теряются при высоком белке
- До видимого нижнего пресса: −2–3 кг жира (с ~15–16% до ~12–13% BF)
- Пресс появится через питание, не через упражнения на кор

_Расчёт: Mifflin-St Jeor, активность 1.55 (зал 2–3×/нед), дефицит 12.5%, Б 2г/кг_
